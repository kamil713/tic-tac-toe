## 	:toolbox: Getting Started

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/kamil713/tic-tac-toe.git
```

Go to the project directory

```bash
  cd tic-tac-toe
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  gulp
```
